Template Layout BookReader

Đây là file layout mẫu để dùng cho các trang động (Thymeleaf, template engine hoặc backend render).
Bạn có thể include header, footer, menu và chèn nội dung động vào phần <main>.
Tham khảo bố cục và màu sắc từ nhasachmienphi.com.

Ví dụ sử dụng layout:

  • Header, menu, footer dùng chung cho mọi trang.
  • Nội dung động sẽ được render vào giữa (giữa thẻ <main>).
  • Thích hợp cho dự án Spring Boot, NodeJS, hoặc bất kỳ backend nào hỗ trợ template engine.

Bạn có thể copy phần header/footer này sang các file HTML khác để đồng bộ giao diện.
